# Privacy Policy

## 1. Scope
- Product: `FinWiseAi`.
- This policy applies to web and PWA clients, API, support chat and billing events.

## 2. Personal Data We Process
- Account data: `phone`, `email`.
- Auth/session metadata: login timestamps, session identifiers.
- Financial data entered by user: accounts, transactions, categories, budget plans.
- Support data: support chat messages.
- Billing and subscription data: tariffs, payment states, subscription periods.

## 3. Purpose of Processing
- Provide core finance and AI assistant features.
- Protect user account and prevent abuse.
- Process subscriptions and payment lifecycle.
- Fulfill legal obligations and security audit requirements.

## 4. Data Minimization
- Only required data is collected.
- Passwords are stored as hashes.
- API secrets and payment secrets are stored only in environment variables.
- Sensitive values are not logged intentionally.

## 5. User Rights and Requests
- User can request correction or deletion via support chat.
- User can request export/deletion review by support/admin team.

## 6. Security Controls
- RBAC (`USER` / `ADMIN`).
- Rate limiting for API.
- Security response headers.
- Audit log for admin actions.
- Payment webhook idempotency.

## 7. Contacts
- Support channel: in-app support chat.
