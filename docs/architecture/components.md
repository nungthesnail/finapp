# Components

## Web app (target split)

- `app/router`: route tree and access guards.
- `app/layout`: shared shells for user/admin/public.
- `pages/*`: screen-level composition.
- `features/*`: use-case focused UI blocks and state.
- `entities/*`: domain UI models and simple adapters.
- `shared/api`: transport, endpoint wrappers, error mapping.
- `shared/ui`: base visual primitives.
